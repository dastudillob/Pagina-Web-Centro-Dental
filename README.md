# Pagina-Web-Centro-Dental

Proyecto de página web para centro dental utilizando HTML, CSS, Boostrap 5 y JavaScript.
Este permite Buscar, desplegar, modificar y eliminar los pacientes de la base de datos.
Generar un reporte con su historial de consultas medicas.
Generar un presupuesto e imprimirlo.
Agendar horas según local, paciente y doctor.
